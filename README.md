# Retail_Store-Data-Analysis-Report-.XL
## summary
This project demonstrates how Excel’s Data processing, cleaning, pivot table can be applied to sales Management. By modeling sales by state, media, gender and age variations, store manager can predict with sales targets.These insights enable better decision‑making in compensation planning.
## Objective
To analyze store sale structures and variations.
To apply Excel Power Query for data cleaning and shaping.
also apply Pivot Table for summarize data, fitering and grouping and interactive Dashboard.
## methodology
Data Cleaning: Data cleaning in power query.
Data Processing: grouping age and find only month from the date 
PivotTables & Charts: Summarized sales trends and visualized outcomes.
## Analysis 
Sample Insights
Women are more likely to buy compared to men (~64%)
Maharashtra, Karnataka and Uttar Pradesh are the top 3
Adult age group (30-49 yrs) is max contributing (~50%)
Amazon, Flipkart and Myntra channels are max contribution
## conclusion
Target women customers of age group (30-49 yrs) living in Maharashtra, Karnataka and Uttar Pradesh by showing ads/offers/coupons available on Amazon, Flipkart and Myntra
